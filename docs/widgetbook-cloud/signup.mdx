# Sign-up

Sign up for Early Access to Widgetbook Cloud by visiting [widgetbook.io](https://www.widgetbook.io/plans).

**New to Widgetbook Cloud?**

- Create a new account by choosing the Sign-up option and providing the necessary information.
- You will be automatically logged in once the sign-up process is complete.

**Existing Widgetbook Cloud user?**

- Sign in using your existing login credentials.

**Forgot your password?**

- Utilize the password recovery feature to reset your password.
- The password recovery link will expire after a few hours for security purposes.
