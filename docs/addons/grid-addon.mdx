# Grid Addon

The Grid Addon is a Widgetbook utility designed for adding a pixel grid
use-case being previewed within the Workbench.

## Setup

Here's how to configure it:

<Warning>
  In case `GridAddon` is used with other addons, then it must be defined before
  the [`AlignmentAddon`](/addons/alignment-addon) and after the
  [`DeviceFrameAddon`](/addons/device-frame-addon)
</Warning>

```dart
GridAddon(10),
```
