# Grid Addon

The Grid Addon is a Widgetbook utility designed for adding a pixel grid
use-case being previewed withing the Workbench.

## Setup

Here's how to configure it:

<Warning>`GridAddon` must be defined before the [`AlignmentAddon`](/addons/alignment-addon)</Warning>

<Warning>When used with `DeviceFrameAddon`, `GridAddon` must be defined after the [`DeviceFrameAddon`](/addons/device-frame-addon)</Warning>

```dart
GridAddon(10),
```
