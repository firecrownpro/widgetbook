## Setup

The CLI is available as the [widgetbook_cli](https://pub.dev/packages/widgetbook_cli) package.

Install the CLI globally by running:

```bash
dart pub global activate widgetbook_cli
```

Run the CLI by using

```bash
widgetbook <command> [arguments]
```

## Push Widgetbook Builds to Widgetbook Cloud

An existing Widgetbook can be uploaded via the `cloud build push` command.
