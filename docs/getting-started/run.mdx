# 3. Run

Check if everything works by running `flutter run` with the newly created Widgetbook:

<Tabs
  values={[
    { label: 'MacOS', value: 'macos' },
    { label: 'Web', value: 'web' },
  ]}
>
  <TabItem value="macos">
  ```bash 
  flutter run -d macos -t lib/app.widgetbook.dart 
  ```
  </TabItem>
  <TabItem value="web">
  ```bash 
  flutter run -d chrome -t lib/app.widgetbook.dart 
  ```
  </TabItem>
</Tabs>
