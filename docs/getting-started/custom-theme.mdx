# Custom Theme

`Widgetbook` allows developers to define whatever `Theme` they have defined for
their app. To accompany every theme and the ones defined by Flutter, the
following constructors exist:

| Theme        | Constructor             | Defaults to                      |

| Cupertino    | `Widgetbook.cupertino`  | `Widgetbook<CupertinoThemeData>` |
